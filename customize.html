Customize and download


Customize Fine Uploader's look, server-type, and components to get your very own version.


Note: A valid email is required to build a custom copy of Fine Uploader. Your email address is automatically added to the list of verified emails after your purchase.

jQuery

Select whether you would like to use the jQuery plugin version with Fine Uploader. The plugin makes it super easy to integrate Fine Uploader into any site already using jQuery.

The jQuery plugin's syntax is a bit different from the non-jQuery uploader. Be sure to read the manual!

Server

Select the service you are using as a backend. Traditional assumes you will be implementing your own upload server, and S3 will provide you with the code to send uploads directly to Amazon's Simple Storage Service.

User-interface

Fine Uploader provides an option "UI-mode" which contains basic UI elements (such as a dynamic list of submitted files, coloring based on status, and more). Select whether you would like your custom build to include UI mode.
